Sec Nav v1.1.0+ HelixUI v1.1.0 or later required

Sec Nav (i.e., secondary navigation) is a variation of navigation that is used in lieu of left navigation. It establishes a user's location within an application. Unless a firm implementation constraint requires Sec Nav, use the left navigation component for web applications.
