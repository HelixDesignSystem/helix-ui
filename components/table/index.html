CSS Classes

.hxTable
Use on a table element to apply Helix styles. This class is required to apply any modified table styling.
.hxTable--bordered
Use along with the .hxTable class to apply Helix styles for bound table data.
.hxTable--condensed
Use along with the .hxTable class to apply Helix styles for condensed data.
.hxTable--hover
Use along with the .hxTable class to apply Helix styles when hovering over a table row.
.hxTable__control-col
Use on a table cell to propery set its width for use with a checkbox, radio, or action icon.
.selected
Use on a table row to indicate selection.
