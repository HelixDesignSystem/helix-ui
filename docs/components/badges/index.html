Each badge should have whitespace before the opening tag in your markup. Otherwise, CSS styles may not apply correctly.

Badges are implemented by defining a <span> with the .hxBadge CSS class applied.

Link Badge

Badges may be used with a hyperlink to bring attention to new items at the destination.

Link 1
{% code 'html' %} Link 1 {% endcode %}

Icon Badge

Badges can be combined with icons to create a compact notification.

1 2 32 99+
{% code 'html' %} 1 2 32 99+ {% endcode %}

Tab Badge

Badges may be used in tabs to bring attention to new items in unselected tabs.

First 1 Second 32 Third 99+ ... ... ...
{% code 'html' %} First 1 Second 32 Third 99+ ... {% endcode %}
