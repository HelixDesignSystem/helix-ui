--- title: minver: 1.0.0 controlElement: also: components/toggle: Toggle elements/hx-toggle: --- {% extends 'element.njk' %} {% import '_utils.njk' as utils %} {% block page_header %}

The {{page.title}} is a container that provides behavior to augment native browser validation capabilities for the custom <hx-toggle> element. It is used to build a Toggle component.

{% endblock %} {% block content %}
Permitted Parents
any element that accepts flow content
Permitted Children
in this order:
  1. checkbox {{utils.element('input')}} element
  2. {{utils.element('label')}} (wrapper) element
    • {{utils.element('hx-toggle')}} (facade) element
Events
none
{% endblock %} {% block attributes %}
hx-changed {Boolean} read-only

Present if descendant {{page.controlElement}} element has emitted a "change" event.

hx-dirty {Boolean} read-only

Present if descendant {{page.controlElement}} element has emitted a "change" or "blur" event.

hx-touched {Boolean} read-only

Present if descendant {{page.controlElement}} element has emitted a "blur" event.

{% endblock %} {% block properties %}
controlElement {?HTMLInputElement} read-only

Returns the first {{page.controlElement}} descendant or null if none are found.

toggle {Boolean [false]} read-only

Returns the toggled state of the <hx-toggle> element.

toggleElement {?HXToggleElement} read-only

Returns the first <hx-toggle> descendant or null if none are found.

isDirty {Boolean [false]} read-only
True if controlElement has emitted a "change" or "blur" event.
wasChanged {Boolean [false]} read-only
True if controlElement has emitted a "change" event.
wasTouched {Boolean [false]} read-only
True if controlElement has emitted a "blur" event.
{% endblock %}
