--- title: minver: 0.2.0 also: components/popovers: Popovers elements/hx-disclosure: elements/hx-div: --- {% extends 'element.njk' %} {% block content %}

The custom {{page.title}} element defines a section of floating content to appear nearby other elements. It is often toggled by a companion <hx-disclosure> element.

Permitted Parents
any
Permitted Children
In this order:
  1. <header> (optional)
  2. <hx-div>
  3. <footer> (optional)
Events
Any of the following:
  • close
  • open
{% endblock %} {% block attributes %}
open
When present, the popover will be revealed.
position
Configures the position of the popover in relation to its relative element.
relative-to (optional)
ID of an element that the popover is positioned relative to.
{% endblock %} {% block properties %}
controlElement (HTMLElement)

(read-only) Returns the HTML element with the aria-controls attribute that matches the popover ID.

open (Boolean [false])

Manipulates the open attribute.

position (String [bottom-right])

Manipulates the position attribute.

relativeElement (HTMLElement)

(read-only) Returns the HTML element used as a reference to calculate the position of the popover. If the relative-to attribute is set, this will be the element with ID matching the relativeTo property value. Otherwise, this will be the controlElement.

relativeTo (String)

Manipulates the relative-to attribute.

{% endblock %}
