The custom <hx-popover> element defines a section of floating content to appear nearby other elements.

Permitted Parents
any
Permitted Children
in this order:
  1. an optional <hx-popover-head> element
  2. an <hx-popover-body> element
  3. an optional <hx-popover-foot> element
Events
Any of the following:
  • close
  • open
{% endblock %} {% block attributes %}
open
When present, the popover will be revealed.
position
Configures the position of the popover in relation to its trigger element.
{% endblock %} {% block properties %}
open (Boolean [false])
Manipulates the open attribute.
position (String [bottom-right])
Manipulates the position attribute.
{% endblock %}
