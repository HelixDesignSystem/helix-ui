The custom {{page.title}} element defines a section of floating content to appear nearby other elements. It is often toggled by a companion <hx-disclosure> element.

Permitted Parents
any
Permitted Children
In this order:
  1. <header> (optional)
  2. <hx-div>
  3. <footer> (optional)
Events
Any of the following:
  • close
  • open
{% endblock %} {% block attributes %}
open
When present, the popover will be revealed.
position
Configures the position of the popover in relation to its trigger element.

controlElement (HTMLElement)

(read-only) Returns the HTML element with the aria-controls attribute that matches the popover ID.

open (Boolean [false])
Manipulates the open attribute.
position (String [bottom-right])
Manipulates the position attribute.
