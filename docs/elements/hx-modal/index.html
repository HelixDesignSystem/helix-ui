The custom <hx-modal> element defines a section of floating content that needs a forced action from the user.

Permitted Parents
any
Permitted Children
in this order:
  1. an optional <hx-modalhead> element
  2. an <hx-modalbody> element
  3. an optional <hx-modalfoot> element
Events
Any of the following:
  • close
  • open
