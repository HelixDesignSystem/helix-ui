The custom {{page.title}} element provides a visual overlay when dragging a file, marking the boundaries of where a user can drop.

Permitted Parents
any
Permitted Children
any
Events
Any of the following:
  • drop - See MDN's "File Drag and Drop" for more information.
{% endblock %} {% block attributes %}
drag {String}
(read-only)

Its value is automatically set by the element itself and mainly used for CSS styling purposes.

{% endblock %} {% block properties %}
drag {String}
(read-only)

A value of "away" means a file drag is detected on the document, away from the drop zone.

A value of "over" means a file drag is detected on the document, over the drop zone.
