The custom {{page.title}} element displays supplemental information related to an element on the page.

Permitted Parents
any
Permitted Children
flow content
Events
Any of the following:
  • close
  • open
  • reposition
{% endblock %} {% block attributes %}
for

The ID of an element in the same document as the {{page.title}} element. The first element with a matching ID is known as the control element for the {{page.title}}.

open (optional)

If present, the {{page.title}} will be shown on screen.

position (optional)

Configures the position of the {{page.title}} in relation to its relative element.

relative-to (optional)

The ID of an element in the same document as the {{page.title}} element and the control element. The first element with a matching ID is known as the relative element for the {{page.title}}.

If omitted, the control element will act as the relative element.

{% endblock %} {% block properties %}
controlElement {HTMLElement} read-only

Returns the HTML element whose ID matches the htmlFor value.

The control element is used to attach various DOM event listeners, in order to determine when to show or hide the {{page.title}}.

htmlFor {String}

A string corresponding to the ID of the control element.

Reflects the for attribute.
