The {{page.title}} block element is a container that provides behavior to augment native browser validation capabilities. It is used to build a Radio component.

{% endblock %} {% block content %}
Permitted Parents
an <hx-radio-set> element
Permitted Children
in this order:
  1. radio {{utils.element('input')}} element
  2. {{utils.element('label')}} element
  3. {{utils.element('p')}} element for control-level help text (optional)
Events
none
{% endblock %} {% block attributes %}

Present if descendant {{page.controlElement}} element has emitted a "change" event.

hx-dirty {Boolean} read-only

Present if descendant {{page.controlElement}} element has emitted a "change" or "blur" event.

hx-touched {Boolean} read-only

Present if descendant {{page.controlElement}} element has emitted a "blur" event.
