The {{page.title}} block element is a container that provides behavior to augment native browser validation capabilities. It is used to build a Radio component.

Permitted Parents
any element that accepts flow content
Permitted Children
in this order:
  • radio {{utils.element('input')}} element
  • {{utils.element('label')}} element
  • {{utils.element('p')}} element for control-level help text (optional)
Events
none
