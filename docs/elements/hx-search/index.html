--- title: minver: 0.4.0 also: components/search: Search elements/hx-search-assistance: --- {% extends 'element.njk' %} {% block page_header %}

The custom {{page.title}} element renders a search control that accepts user input.

{% endblock %} {% block content %}
Permitted Parents
any
Permitted Children
none (content will be removed upon element upgrade)
Events
Any of the following:
  • blur (doesn't bubble)
  • clear on clear
  • focus (doesn't bubble)
  • input as user types

Methods

focus()
Overrides default HTMLElement behavior to focus the input in the Shadow DOM.
clear()
Simulates pressing "x" to clear the value of the input.
{% endblock %} {% block attributes %}
disabled
Disables user interaction
invalid
Marks input as invalid
placeholder
Hint of what can be entered into the control
value
Initial value of the control
{% endblock %} {% block properties %}
disabled {Boolean [false]}
Get / Set the disabled state of the control.
invalid {Boolean [false]}
Get / Set the invalid state of the control.
placeholder {String}
Get / Set the control's placeholder text. The placeholder text must not contain carriage returns or line-feeds
value {String}
Get / Set the current value of the control.
