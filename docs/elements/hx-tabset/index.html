--- title: minver: 0.2.0 also: components/tabs: Tabs elements/hx-tab: elements/hx-tabcontent: elements/hx-tablist: elements/hx-tabpanel: --- {% extends 'element.njk' %} {% block page_header %}

The custom {{page.title}} element provides functionality to navigate between one or more tab panels.

{% endblock %} {% block content %}
Permitted Parents
any
Permitted Children
one of each:
  • an <hx-tablist> element
  • an <hx-tabcontent> element
Events
Any of the following:
  • tabchange when the current tab changes

Methods

selectNext()
Selects the next tab in the tabset. If the last tab is currently active, selects the first tab.
selectPrevious()
Selects the previous tab in the tabset. If the first tab is currently active, selects the last tab.
{% endblock %} {% block attributes %}
current-tab

Zero-based index of the currently active tab.

{% endblock %} {% block properties %}
currentTab {Integer [0]}

Zero-based index of the currently active tab.

Reflects the current-tab attribute.

{% endblock %}
