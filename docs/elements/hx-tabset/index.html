The custom {{page.title}} element provides functionality to navigate between one or more tab panels.

{% endblock %} {% block content %}
Permitted Parents
any
Permitted Children
one of each:
  • an <hx-tablist> element
  • an <hx-tabcontent> element
Events
Any of the following:
  • tabchange when the current tab changes

Managing State

Keeping logical and visual states in sync can be tricky.

Logical state should be considered the source of truth when updating visual state. Logical state is defined by attributes and properties of {{page.title}}, while visual state is defined by the appearance and configuration of its children.

  • If the logical state is incorrect, updating the current-tab attribute should be enough to fix the inconsistency.
  • In all other cases, calling the update() method on {{page.title}} should fix visual inconsistencies.

Methods

selectNext()

Selects the next tab in the tabset.

Selects the first tab, if the last tab is currently active.

selectPrevious()

Selects the previous tab in the tabset.

Selects the last tab, if the first tab is currently active.

update()

Synchronize visual appearance with element configuration.

Useful when <hx-tab> and <hx-tabpanel> elements are added/removed dynamically.
