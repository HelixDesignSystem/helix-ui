position :PositionString

Where to position the element against its relativeElement.

NOTE: Values are normalized upon setting, which means that the value retrieved may differ from the value being set.

el.position = 'top';
el.position; // 'top-center'

el.position = 'bottom-center';
el.position; // 'bottom-center'
